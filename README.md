installation guide:

1.simply download source sdk 2013 rtx mod and extract it in sdk 2013 folder. 

2.once you have done that open source sdk 2013 to test it and see if it works if it does it should look like this https://imgur.com/a/yG9HwbM also ignore the sourcevr error that happens to me too and it doesnt make issues with the rtx remix modding

3.if you wanna play your mod you gotta simply drag YOURMOD into source sdk 2013 single player directory

4.after you do that make sure in the start.bat right after hl2.exe in the section called -game YOURMODNAME replace the yourmodname with your mod. here is image if you feel confused about it what to replace https://imgur.com/a/ymUnV9F

5.after that just launch the start.bat and it should be good to go.

note:sometimes in start.bat if it doesnt work with -dxlevel 90 or +mat_dxlevel 90 try replacing the number 90 with either 80,70 or 60 and see which one will it work i had to do that myself and got few mods working.
